# RestaurantPOS Frontend

Frontend app untuk RestaurantPOS berbasis Expo + React Native.

## Setup

1. Install dependency:

```bash
npm install
```

2. Salin template environment:

```bash
cp .env.example .env
```

3. Jalankan aplikasi:

```bash
npm start
```

## Scripts

- `npm start` menjalankan Expo
- `npm run android` menjalankan build Android
- `npm run web` menjalankan web preview
- `npm run test:minimal:frontend` menjalankan test frontend minimal

## Catatan Repository

- Repository ini disiapkan untuk frontend saja.
- File sensitif seperti `.env`, log, cache, build output, dan material sertifikat tidak ikut di-commit.
- Gunakan `.env.example` sebagai acuan konfigurasi lokal.
